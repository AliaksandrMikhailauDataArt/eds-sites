04 — Usage Guidelines
DESIGN RULES
Core principles for maintaining visual consistency across the Tatry Topo platform.

Color Hierarchy

Use granite-900 for primary text and dark backgrounds. summit-orange is the primary action color — reserve it for CTAs, active states, and key data. Never use more than two accent colors in a single view.

Grade Colors Only for Grades

The five grade colors (green → red → purple) must only be used to represent UIAA difficulty ratings. Never use grade colors for unrelated UI elements — this prevents confusion in topo diagrams.

Typography Pairing

Bebas Neue for hero/display only (never body text). Barlow Condensed for headings and UI labels. Barlow for descriptions and body. JetBrains Mono for coordinates, grades, and technical data.

Topo Diagram Lines

Route lines on topo photos use grade colors with 3px stroke and drop-shadow for contrast. Belay stations are 8px circles with white stroke. Use dashed lines for approach/descent paths.

✓ Do

  • Use generous whitespace between route cards
  • Display grade badges with consistent sizing
  • Use dark mode for topo photo overlays
  • Pair Bebas Neue with Barlow Condensed
  • Use granite-900 for primary backgrounds
  • Keep grade colors reserved for difficulty

✕ Don't

  • Use summit-orange for error states (use ridge-red)
  • Mix grade colors for decorative purposes
  • Use Bebas Neue below 20px
  • Set body text in Barlow Condensed
  • Use more than 2 accent colors per view
  • Place light text on light granite backgrounds
